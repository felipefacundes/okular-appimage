# okular-appimage
okular appimage finally the best pdf reader in AppImage . Lovingly crafted AppImage tools and resources . Okular is a multiplatform document viewer developed by the KDE community and based on Qt and KDE Frameworks libraries. It is distributed as part of the KDE Applications bundle. Its origins are from KPDF and it replaces KPDF, KGhostView, KFax, KFaxview and KDVI in KDE 4.
